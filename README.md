# FocusFlow (iOS)

FocusFlow is a calm, elegant focus + habit app built with SwiftUI. It helps you run focus sessions, build streaks, track progress, and personalize your profile with avatars and themes. The app supports Guest Mode (local-only) and optional secure cloud sync via Supabase when signed in.

## Status
This repository is not intended as a reusable template or starter kit.

## Features
- Focus sessions with intentions, presets, sounds/haptics, and Live Activities (iOS 16+)
- Habit tracking with reminders, repeat schedules, and progress UI
- Stats, achievements, and leveling
- Profile customization (SF Symbol avatars — no photo permissions required)
- Optional cloud sync for sessions and stats (Supabase)
